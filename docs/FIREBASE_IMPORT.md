# Импорт данных из Firebase (legacy)

1. Экспортируйте события из Firestore (JSON) и при необходимости свяжите поле `name` с `legacy_name` в таблице `events`.
2. Заполните `events.legacy_name` тем же текстом, что использовал клиент как ключ в Realtime DB, чтобы сохранить маппинг для отладки.
3. Запустите `python scripts/seed_data.py` для базового набора или добавьте скрипт импорта поверх миграций Alembic.

API и мобильное приложение опираются на стабильный UUID `events.id`, отображаемый в клиенте как `serverId`.
